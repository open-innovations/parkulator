Parkulator (static)

Parkulatable

Notes

This tool finds polygons (ways) in OpenStreetMap for the selected feature type and then calculates how many parks and how many homes you'd be able to have on the same footprint.

To use it, click the at the top-left of the map, draw an area, then click the "Parkulate!" button.

If you notice omissions, go to OpenStreetMap and check if the feature exists and if it has been tagged correctly. Updates made to OpenStreetMap should filter through to this tool within a few hours.

If you have a simple single Polygon boundary in a GeoJSON file then you can just drag-and-drop the file onto the map and the outline will be added for you. In case it is helpful, we have GeoJSON files for UK Local Authority Districts in our Geography Bits repository.
